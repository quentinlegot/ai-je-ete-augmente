# ai-je-ete-augmente

A small single page app to show the difference between your usual salary raises, and the reality with your income adjusted with the inflation rates.

## Planned feature

* [x] English translation
* [ ] Use data from eurostats to enable other country than france
* [ ] Add new Raise calculation mode with % increae instead of flat value
* [ ] Add indice point calculation mode

## How to contribute?

If you can, feel free to open a pull request. Else, you can open an issue.

## Development

See [Vite Configuration Reference](https://vitejs.dev/config/).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Continuous Integration checks

```sh
npm run format && npm run lint && npm run type-check
```
